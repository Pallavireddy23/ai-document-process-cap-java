# AI-Powered Document Processing System

## Overview
Cloud-native document processing service using CAP Java, SAP BTP, and AI.

## Tech Stack
- CAP Java
- Python (OCR + NLP)
- SAP UI5
- OData APIs
- Docker
- SAP BTP (Cloud Foundry)

## Features
- Document upload (PDF/Image)
- OCR text extraction
- AI-based key-value extraction
- REST/OData APIs
- Enterprise UI dashboard

## Architecture
Backend (CAP Java) → AI Service (Python) → UI (SAP UI5)

## How to Run
1. Run CAP service
2. Start Python AI service
3. Launch UI5 frontend
